# 留学申请管理系统 (Study Abroad Manager) 需求文档

## 1. 项目简介
本项目是一个面向留学生的申请管理工具，旨在通过直观、简洁的界面（类 Notion 风格）帮助用户追踪多个留学项目的申请进度、管理任务及对比各项要求。

## 2. 核心功能

### 2.1 项目管理 (Project Management)
- **添加项目**：用户可以添加申请的学校名称及专业。
- **项目概览**：以卡片形式展示所有申请项目，包含学校、专业、基本要求预览及专属任务预览。
- **进度追踪**：每个项目卡片配有进度条，自动计算（通用任务 + 项目专属任务）的完成百分比。
- **删除项目**：支持移除不再考虑的申请项目，并同步清理相关任务。

### 2.2 申请要求配置与对比 (Requirements & Comparison)
- **要求配置**：为每个项目单独配置详细要求，包括：
    - 语言成绩 (IELTS/TOEFL)
    - 推荐信数量 (Recommendation Letters)
    - 个人陈述 (PS) 是否需要
    - 研究计划 (RP) 是否需要
- **横向对比**：提供专门的对比页面，以表格形式清晰展示不同项目间的各项要求差异。

### 2.3 任务管理系统 (Task System)
- **任务分类**：
    - **通用任务**：所有项目共享的任务（如：准备通用 CV）。
    - **专属任务**：绑定到特定项目的任务（如：联系 UCL 的推荐人）。
- **任务属性**：
    - 标题 (Title)
    - 详情/概要 (Description)
    - 截止日期 (Deadline/DDL)
    - 完成状态 (Completion Status)
- **操作支持**：
    - **添加任务**：支持设置标题、详情、日期及所属项目。
    - **修改任务**：支持对已存在任务的所有属性进行编辑。
    - **删除任务**：支持移除任务。
    - **标记完成**：支持通过勾选框实时切换完成状态，并联动更新项目进度。

## 3. 界面与交互 (UI/UX)
- **风格**：简洁的 Notion 风格，使用纯白色背景、灰色边框及 Lucide 图标库。
- **侧边栏导航**：支持“项目概览”、“任务中心”、“要求对比”快速切换。
- **响应式设计**：适配不同屏幕尺寸，保证在移动端和 PC 端都有良好的浏览体验。
- **弹窗交互**：使用 Modal 弹窗进行添加和修改操作，保持主页面的整洁。

## 4. 技术实现
- **前端技术**：HTML5, CSS3 (Tailwind CSS), JavaScript (Vanilla JS)。
- **图标库**：Lucide Icons。
- **数据存储**：使用浏览器 `localStorage` 进行本地持久化存储，无需后端数据库。
- **版本控制**：托管于 GitHub，支持 Git 流程同步。
